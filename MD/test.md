# ZhiPath 用户完整业务流程

> 视角：大三计算机专业学生，想找前端开发工作
> 一句话总结：用户来这里不是"找工作"，而是"变强了再找工作"——学习是手段，匹配是结果，投递是终点。

---

## 第一幕：首次打开（注册/登录）

1. 打开浏览器 → 进入 `/login` 页面
2. 注册账号（手机号/邮箱 + 密码）
3. 登录成功 → 系统判断是新用户 → 跳转 `/user/onboarding`

---

## 第二幕：Onboarding 引导（首次登录专属）

页面加载后，右下角 AI 浮窗自动弹出：

> "你好！我是智途 AI 助手，先了解一下你的情况吧"

### AI 对话流程

| AI 提问 | 用户回答 |
|---------|---------|
| 你是什么专业的？ | 计算机科学与技术 |
| 大几了？什么时候毕业？ | 大三，2027年毕业 |
| 你对哪个方向感兴趣？前端、后端、还是其他？ | 前端开发 |
| 你目前会什么技术？ | HTML/CSS/JavaScript 基础，学过一点 React |
| 每天能花多少时间学习？ | 大概 2 小时 |
| 有目标岗位吗？ | 想进大厂做前端 |

### AI 分析完毕

- 调用 **PlannerAgent** → 生成第一个学习计划
- 弹出确认：

```
我为你生成了「前端开发工程师」学习计划：
├── 阶段1：HTML/CSS/JS 巩固（2周）
├── 阶段2：React + TypeScript（4周）
├── 阶段3：工程化 + 项目实战（3周）
└── 预计 9 周后可匹配初级前端岗位

[开始学习]  [调整计划]
```

> **系统后台**：Onboarding 对话内容自动写入个人中心（专业、年级、技能、方向），同时触发 ResourceAgent 为计划中的技能点异步生成学习资源。

---

## 第三幕：5分钟速测（冷启动正反馈）

点击 [开始学习] 后，系统弹出：

> "来个 5 分钟速测，看看你的基础怎么样？"
> [开始速测]  [跳过，直接学习]

- 选择 [开始速测] → 5道 JavaScript 基础题 → 全对

- **"自报技能"升级为"测试确认技能"**（信任度从 0.3 → 1.0）

## 完整循环总结

```
注册 → Onboarding(AI对话) → 生成计划 → 速测(冷启动)
  ↓
主页(今日任务) → 学习(讲义/视频/练习/编程) → 技能掌握度↑
  ↓
阶段考试 → 技能确认 → 匹配度↑
  ↓
岗位匹配 → 差距分析 → 补充学习 → 匹配度达标
  ↓
生成简历 → 投递 → 管理员审核 → 发送企业
  ↓
持续学习 → 换目标/加支线 → 循环
```

## 涉及页面路由

| 路由 | 页面 | 说明 |
|------|------|------|
| `/login` | 登录/注册 | 首次入口 |
| `/user/onboarding` | 引导对话 | 新用户专属，AI 对话收集信息 |
| `/user/home` | 主页 | 日常起点，今日任务 + 数据看板 |
| `/user/learning/:id` | 学习详情 | 讲义/视频/练习/编程题 |
| `/user/exams` | 考试列表 | 阶段考试入口 |
| `/user/exams/:id/take` | 答题页 | 计时答题 |
| `/user/jobs` | 岗位列表 | 匹配度排序 + 筛选 |
| `/user/jobs/:id` | 岗位详情 | 差距分析 + 提升建议 |
| `/user/agent-office` | 智能体办公室 | Agent 状态 + 任务队列 |
| `/user/progress` | 学习记录 | Git 风格日志 + 雷达图对比 |
| `/user/resume` | 简历管理 | 生成/编辑/导出/投递 |

## 涉及智能体

| Agent | 职责 | 触发时机 |
|-------|------|---------|
| **PlannerAgent** | 生成/调整学习计划 | Onboarding、换目标岗位、加入新技能 |
| **ResourceAgent** | 生成学习资源（讲义/练习题） | 计划生成后异步触发、用户标记紧急 |
| **MatchAgent** | 计算岗位匹配度 | 技能更新时增量计算 |
| **ReviewerAgent** | 审核简历 | 简历投递后自动触发 |
| **ResumeAgent** | 生成简历 | 学完阶段2后解锁 |
