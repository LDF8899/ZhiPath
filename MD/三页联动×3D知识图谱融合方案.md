## 一、概述

```
传统系统：学习 → 进度条 → 结束
智途系统：拖 Agent → 执行任务 → commit → 图谱变形 → 能力成长

智能体拖拽 = 核心交互（怎么触发）
智能体办公室 = Agent 的家（管理 + 监控 + 详情）
三页联动   = 事件管道（数据怎么通）
3D 图谱    = 可视化（能力怎么呈现）

四者合一 = 智途的独特体验
```

### 为什么用 3D 知识图谱替代雷达图

| 维度 | 雷达图 | 3D 知识图谱 |
|------|--------|------------|
| 技能数量 | 受限于轴数（6-8个） | 无限，节点自动布局 |
| 技能关系 | 无 | 边连接：前置依赖 / 同类关联 / 学习路径 |
| 掌握度 | 轴长度 | 节点大小 + 发光强度 + 颜色饱和度 |
| 时间维度 | 需要多图对比 | 时间滑块，实时变形动画 |
| 分类 | 需要颜色区分 | 3D 空间自动聚类（前端一团、后端一团） |
| 交互 | 点击轴 | 旋转/缩放/点击节点/悬停高亮关联链 |
| 沉浸感 | 平面图表 | 粒子效果、光晕、连接线流动 |

## 二、现状与差距

### 2.1 三页断裂点

```
Chat 生成学习路径 → Learning 不知道 → 用户手动刷新
Learning 完成技能 → Chat 不知道 → 无鼓励反馈、匹配度不更新
AgentOffice 任务完成 → 仅页面内可见 → Chat/Learning 无感知
```

### 2.2 AgentOffice 现状 vs 目标

```
✅ 已有的                              ❌ 缺少的
─────────────────────────────────────────────────────
✅ 等距风格办公室场景                   ❌ 中央任务面板（实时进度）
✅ 工位拖拽（Agent 换工位）             ❌ 拖 Agent 到 Chat 页面
✅ 任务队列 + 拖拽排序                  ❌ Agent 详情面板（能力覆盖/统计）
✅ 招聘/编辑/解雇弹窗                   ❌ Agent 统计图表
✅ 直接使用（输入 prompt）              ❌ 缩略版 3D 图谱
✅ SSE 事件监听                         ❌ 拖 Agent 到图谱节点
✅ 8 秒轮询刷新                         ❌ workspace store 统一
                                       ❌ AGENT_CONFIGS 双轨制没统一
```

**核心问题：** 1053 行单文件 + 本地 state + 双轨 Agent 配置。改造而不是重写。

### 2.3 已有基础设施（可复用）

| 组件 | 状态 | 复用方式 |
|------|------|----------|
| `useSSE` hook | ✅ 已有，AgentPanel 和 Dashboard 在用 | 扩展到 Learning 页 |
| `EventsService` | ✅ 已有，内存 pub/sub | 补充缺失的 emit 调用 |
| `NotificationService` | ✅ 已有，5 个便利方法无人调用 | 在关键节点触发 |
| `useOfficeStore` | ✅ 已有，Chat+AgentPanel 共用 | 补充 workspace 共享字段 |
| `useChatStore` | ✅ 已有 | 不改结构，加事件发射 |

### 2.4 核心差距

- **后端**：AgentOffice 的 `executeTask` 不发 SSE 事件
- **后端**：`NotificationService` 的便捷方法无人调用
- **后端**：SSE 和 Notification 两个系统完全断开
- **前端**：Learning 页没有任何 store，纯 useState，切页即丢
- **前端**：无事件总线，页面间无法感知对方动作
- **前端**：学习数据只有进度条，无能力可视化
- **前端**：AGENT_CONFIGS 双轨制（intent-based vs backend-based 互不认识）

## 三、核心交互：智能体拖拽

> **这是智途区别于所有竞品的核心玩法。**
> 用户不是"点按钮触发功能"，而是"把智能体拖到需要的地方"。

### 3.1 交互模型

```
┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│   智能体办公室                    AI 聊天                            │

│   │  │学习│  ┌────┐   拖拽       │  用户: 帮我学 React       │      │
│   │  │规划│  │考试│  ──────►     │                          │      │
│   │  │Agent│  │辅导│             │  ┌────────────────────┐  │      │
│   │  └────┘  │Agent│             │  │ 🤖 学习规划Agent    │  │      │
│   │          └────┘             │  │ 正在分析你的技能图谱...│  │      │
│   │  ┌────┐  ┌────┐             │  └────────────────────┘  │      │
│   │  │简历│  │视频│             │                          │      │

│   │  │Agent│  │Agent│             │  React基础 → Hooks → ...  │      │
```

### 3.2 拖拽场景定义

| 拖拽源 | 拖拽目标 | 触发行为 | 图谱响应 |
|--------|----------|----------|----------|
| 学习规划 Agent | Chat 区域 | 分析用户画像，生成个性化学习路径 | 路径上的节点连线发光 |
| 考试辅导 Agent | Chat 区域 | 根据当前技能薄弱点出题 | 薄弱节点高亮闪烁 |
| 考试辅导 Agent | 图谱某个节点 | 针对该技能出专项考试 | 单节点聚焦放大 |
| 简历优化 Agent | Chat 区域 | 分析画像生成简历建议 | 匹配度指标跳动 |
| 视频生成 Agent | 图谱某个节点 | 为该技能生成教学视频 | 节点出现"生成中"光效 |
| 通用 Agent | Chat 区域 | 自由对话 | 无直接响应 |
| 任意 Agent | 学习路径卡片 | 成为该路径的专属辅导 Agent | 路径卡片显示 Agent 头像 |

### 3.3 拖拽的三种落地模式

```
模式 A：拖到 Chat（最常用）
┌─────────────────────────────────────────┐

│      ↓                                  │

│      ↓                                  │

模式 B：拖到图谱节点（精准操作）
┌─────────────────────────────────────────┐

│      ↓                                  │
│  弹出快捷菜单:                           │

模式 C：拖到学习路径（长期绑定）
┌─────────────────────────────────────────┐

│      ↓                                  │
```


### 3.6 拖拽 × 三页联动事件

```typescript
// 拖拽触发的事件
export type WorkspaceEvent =
  // ... 已有事件 ...
  | {
      type: 'agent_dispatched';       // Agent 被拖拽派遣
      agentType: string;
      target: 'chat' | 'node' | 'path';
      targetId?: string;              // 节点ID 或 路径ID
      autoMessage?: string;           // 自动生成的消息
    }
  | {
      type: 'agent_bound_to_path';    // Agent 绑定到学习路径
      agentType: string;
      pathId: number;
    };
```

```
拖拽事件流:
Agent 被拖到 Chat
    ├── emit agent_dispatched { agentType, target: 'chat' }
    ├── Chat 自动发送 "@学习规划Agent 分析图谱..."
    ├── Office 中 Agent 状态 → working
    ├── AgentPanel 中显示工作中
    ├── SSE: agent_status working
    │
    ├── Agent 完成任务
    │   ├── SSE: agent_status idle
    │   ├── SSE: profile_updated (如果影响了技能)
    │   ├── emit agent_task_completed
    │   ├── 图谱变形动画
    │   ├── toast "🤖 Agent 完成: 学习路径已生成"
    │   └── Learning 页自动刷新
    │
    └── 用户切到 Learning 页 → 路径已在那里
```

```typescript
interface SkillGraphNode {

  decayRate: number;             // 衰减速率

  // ── 图谱专用 ──
  position: [number, number, number];  // 3D 位置（布局算法计算）

  cluster: string;                     // 所属聚类
}
```

### 5.2 技能关系 (SkillEdge)

```typescript
interface SkillEdge {
  from: string;                  // 起始技能 ID
  to: string;                    // 目标技能 ID
  type: 'prerequisite' | 'related' | 'path';
  strength: number;              // 关联强度 0-1
}
```

### 5.3 图谱快照 (GraphSnapshot)

```typescript
interface GraphSnapshot {
  timestamp: number;
  commitId: string;
  branchId: string;

  // ── 图谱数据 ──
  nodes: SkillGraphNode[];
  edges: SkillEdge[];

  // ── 聚合指标 ──
  metrics: {
    overallScore: number;        // 综合能力分 (0-100)
    matchScore: number;          // 匹配度
    depthScore: number;          // 深度 (单技能最高掌握度)
    breadthScore: number;        // 广度 (已覆盖类别数)
    balanceScore: number;        // 均衡度
    learningSpeed: number;       // 学习速度 (commit/天)
    consistency: number;         // 连续学习天数
  };

  // ── 变化量 ──
  delta: GraphDelta;
}
```

```typescript
  // ── 共享数据 ──
  matchScore: number;
  agentBusyCount: number;
  agentIdleCount: number;
  unreadNotificationCount: number;

  // ── 图谱数据 ──
  currentSnapshot: GraphSnapshot | null;

  // ── 事件系统 ──
  lastEvent: WorkspaceEvent | null;
  eventSeq: number;              // 单调递增，确保 subscribe 能检测到同类型事件

  // ── Actions ──
  emit: (event: WorkspaceEvent) => void;
  setMatchScore: (score: number) => void;
  setAgentCounts: (busy: number, idle: number) => void;
  setUnreadCount: (count: number) => void;
  applySnapshot: (snapshot: GraphSnapshot) => void;
}

export const useWorkspaceStore = create<WorkspaceState>((set) => ({
  matchScore: 0,
  agentBusyCount: 0,
  agentIdleCount: 0,
  unreadNotificationCount: 0,
  currentSnapshot: null,
  lastEvent: null,
  eventSeq: 0,

  emit: (event) => set((s) => ({
    lastEvent: event,
    eventSeq: s.eventSeq + 1,
  })),

  setMatchScore: (score) => set({ matchScore: score }),
  setAgentCounts: (busy, idle) => set({ agentBusyCount: busy, agentIdleCount: idle }),
  setUnreadCount: (count) => set({ unreadNotificationCount: count }),
  applySnapshot: (snapshot) => set({ currentSnapshot: snapshot }),
}));
```

```typescript
import { useEffect, useRef } from 'react';
import { useWorkspaceStore } from '../stores/workspace';
import { useGraphAnimation } from './useGraphAnimation';
import { toast } from 'react-hot-toast';

export function useWorkspaceSync() {
  const seqRef = useRef(0);
  const animateDelta = useGraphAnimation();

  useEffect(() => {
    return useWorkspaceStore.subscribe(
      (state) => state,
      (state) => {
        if (state.eventSeq <= seqRef.current || !state.lastEvent) return;
        seqRef.current = state.eventSeq;

        const e = state.lastEvent;
        switch (e.type) {
          case 'path_generated':
            toast.success(`📚 学习路径「${e.planName}」已生成（${e.totalSkills}个技能）`);
            break;

          case 'skill_completed':
            toast.success(`🎉 技能「${e.skillName}」学习完成！`);
            useWorkspaceStore.getState().applySnapshot(e.snapshot);
            animateDelta(e.delta);
            if (e.newMatchScore !== undefined) {
              toast(`📈 匹配度更新为 ${e.newMatchScore}%`, { icon: '🎯' });
            }
            break;

          case 'exam_completed':
            if (e.passed) {
              toast.success(`✅ 「${e.skillName}」考试通过！得分 ${e.score}`);
            } else {
              toast(`📝 「${e.skillName}」考试未通过（${e.score}分），加油！`, { icon: '💪' });
            }
            break;

          case 'match_updated':
            toast(`🎯 匹配度更新为 ${e.newScore}%`, {
              icon: e.newScore >= 80 ? '🚀' : '📈',
            });
            break;

          case 'agent_task_completed':
            toast.success(`🤖 Agent 任务完成：${e.agentType}`);
            if (e.delta) animateDelta(e.delta);
            break;

          case 'agent_task_started':
            toast(`⚡ Agent 开始工作：${e.agentType}`, { icon: '🔄' });
            break;

          case 'resource_ready':
            toast.success(`📄 资源已就绪：${e.skillName}（${e.contentType}）`);
            break;

          case 'today_tasks_refresh':
            break; // 静默事件
        }
      }
    );
  }, []);
}
```

```tsx
// SkillEdge.tsx
function SkillEdge({ edge, nodes, animated }) {
  const fromNode = nodes.find(n => n.id === edge.from);
  const toNode = nodes.find(n => n.id === edge.to);

  const points = useMemo(() => {
    const curve = new QuadraticBezierCurve3(
      new Vector3(...fromNode.position),
      new Vector3(
        (fromNode.position[0] + toNode.position[0]) / 2,
        (fromNode.position[1] + toNode.position[1]) / 2 + 0.3,
        (fromNode.position[2] + toNode.position[2]) / 2,
      ),
      new Vector3(...toNode.position),
    );
    return curve.getPoints(50);
  }, [fromNode, toNode]);

  const bothMastered = fromNode.mastery > 50 && toNode.mastery > 50;

  return (
    <group>
      <Line
        points={points}
        color={bothMastered ? '#4ade80' : '#374151'}
        lineWidth={edge.strength * 2}
        transparent
        opacity={bothMastered ? 0.8 : 0.3}
      />
      {animated && bothMastered && (
        <ParticleFlow curve={points} color="#4ade80" speed={0.5} />
      )}
    </group>
  );
}
```

```typescript
// hooks/useForceLayout.ts
function useForceLayout(nodes: SkillGraphNode[], edges: SkillEdge[]) {
  const [positions, setPositions] = useState(nodes.map(n => n.position));

  useEffect(() => {
    const sim = forceSimulation(nodes)
      .force('charge', forceManyBody().strength(-200))
      .force('link', forceLink(edges).distance(2))
      .force('center', forceCenter())
      .force('cluster', forceCluster(nodes))
      .force('y', forceY().y(n => n.category === '基础' ? -2 : 0));

    sim.on('tick', () => {
      setPositions(sim.nodes().map(n => [n.x, n.y, n.z]));
    });

    return () => sim.stop();
  }, [nodes, edges]);

  return positions;
}
```

```typescript
// hooks/useGraphAnimation.ts
export function useGraphAnimation() {
  return useCallback((delta: GraphDelta) => {
    // 1. 节点脉冲（从小变大再回弹）
    delta.nodeChanges.forEach(change => {
      animatePulse(change.nodeId, { scale: [1, 1.3, 1], duration: 600 });
    });

    // 2. 新边渐入（灰色→绿色，延迟 400ms）
    delta.newEdges.forEach(edge => {
      animateEdgeAppear(edge.from, edge.to, { duration: 800, delay: 400 });
    });

    // 3. 新节点弹入（从无到有，延迟 200ms）
    delta.newNodes.forEach(nodeId => {
      animateNodeAppear(nodeId, { scale: [0, 1], duration: 500, delay: 200 });
    });

    // 4. 位置微调（力导向重新计算，平滑过渡 1 秒）
    recalculateLayout({ transition: 1000 });
  }, []);
}
```

```typescript
async function commitSkill(planId: string, skillName: string, message?: string) {
  const plan = await getPlan(planId);
  const user = await getUser(plan.userId);

  // 1. 更新技能掌握度
  const skill = findSkill(user.skills, skillName);
  const oldMastery = skill?.mastery || 0;
  const newMastery = calculateNewMastery(skill, learningData);
  await updateUserSkill(user.id, skillName, newMastery);

  // 2. 生成图谱快照
  const snapshot = await generateGraphSnapshot(user.id);

  // 3. 计算变化量
  const previousSnapshot = await getLatestSnapshot(user.id);
  const delta = calculateGraphDelta(previousSnapshot, snapshot);

  // 4. 创建 commit（携带快照）
  const commit = await saveCommit({
    branchId: getCurrentBranch(plan).id,
    type: 'skill-complete',
    skillName,
    message: message || `完成 ${skillName}`,
    snapshot,
    delta,
  });

  // 5. 保存快照
  await saveSnapshot(snapshot);

  // 6. 更新匹配度
  const newMatchScore = await calculateMatchScore(user.id, plan.targetJobId);
  await updateMatchScore(user.id, newMatchScore);

  // 7. 🔴 发射富 SSE 事件
  this.eventsService.emitProfileUpdate(user.id, {
    skillName,
    delta,
    snapshot,
    newMatchScore,
  });

  // 8. 🔴 触发通知
  this.notificationService.notifyMatchChange(user.id, plan.targetJobTitle, newMatchScore);

  return { commit, snapshot, delta, newMatchScore };
}
```
