## 今日完成事项

### 1. 学习深度改进 — Prompt 全量重写

**问题**：智能体输出的学习资料太肤浅，用户在学习页面学不到东西。讲义是百科词条式模板，代码是展示品不是练习，拓展阅读只是摘要目录。

**方案**：`MD/学习深度改进方案_20260614.md`

#### LectureAgent — 教学式讲义 prompt

| 维度 | 旧 | 新 |
|------|-----|-----|
| 开头 | "XX 是一种..."定义式 | 真实痛点/故事引入 |
| 结构 | 概述→前置→概念→代码→误区→总结 | 痛点→为什么需要→核心直觉→概念→动手验证→深入→陷阱→思考题 |
| 类比 | 碎片化 | 贯穿全文一个类比 |
| 互动 | 无 | 思考暂停 + 开放思考题 |
| 代码 | 展示 | before/after 对比 + "如果改错会怎样" |
| 题型 | 选择题 3-5 道 | 5-7 道混合（选择 + 概念反转 + 迁移 + 编程） |
| 字数 | 2000-4000 | 3000-5000 |

#### CodeAgent — 渐进式练习 prompt

| 维度 | 旧 | 新 |
|------|-----|-----|
| 输出 | 3 个完整代码片段 | 3 个填空式练习 |
| 结构 | code + comments + output | setup(80%) + task(20%) + hint + solution + solutionExplanation |
| 渐进 | 无关联 | 练习 1 基础 → 练习 2 变体 → 练习 3 综合 |
| 错误 | 无 | commonMistakes（每个练习） |

#### ReadingAgent — 深度短文 prompt

| 维度 | 旧 | 新 |
|------|-----|-----|
| 输出 | 标题 + 100-200 字摘要 | 800-1200 字完整文章 |
| 类型 | 无区分 | why / practice / deep / compare |
| 开头 | "本文将介绍..." | 故事、反直觉事实、具体问题 |
| 结构 | 摘要 + 关键词标签 | 有观点有论据有代码有结尾思考 |

#### Quiz 题型增强

- 解析器新增 `概念反转题` 和 `迁徙题` 类型识别
- 后端 `lecture-agent.service.ts` + 独立 `generate-lecture.ts` 同步更新

### 2. 前端交互改造
